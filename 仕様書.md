## 1. 概要

### 1.1 ゲーム概要
「デスノート人狼」のオンラインマルチプレイヤー版。4〜8人でプレイ可能。キラ陣営、L陣営、第3陣営（メロ）が正体を隠しながら勝利条件を目指すカードゲーム。

### 1.2 技術スタック
| カテゴリ | 技術 |
|---------|------|
| Frontend | React, Tailwind CSS, Framer Motion |
| Backend | Node.js, Socket.io |
| State | Redux Toolkit または Context API |
| Language | TypeScript |
| Deployment | Render |

### 1.3 対応プラットフォーム
- PC（推奨）
- スマートフォン（レスポンシブ対応）

## 3. 役職と能力

### 3.1 キラ陣営

| 役職 | 能力 | 番号変更可能カード |
|------|------|------------------|
| キラ（夜神月） | デスノート所持時、裁きの時間に必ず1人殺害 | デスノート(0) |
| 信者（弥海ミサ） | 裁きの時間・死神カード使用時にキラを視認、専用チャット可能 | なし |

### 3.2 L陣営

| 役職 | 能力 | 番号変更可能カード |
|------|------|------------------|
| L | 逮捕カードを使用可能 | 逮捕(1) |
| 警察（夜神総一郎等） | 拳銃カードを使用可能 | 拳銃(2) |
| ワタリ | ゲーム開始前にLが誰か把握 | 逮捕(1) |

### 3.3 第3陣営

| 役職 | 能力 | 番号変更可能カード |
|------|------|------------------|
| メロ | 拳銃でプレイヤーを殺害可能、キラ殺害で単独勝利、使用後正体公開 | なし |

### 3.4 番号変更能力の詳細

特定の役職は、特定のカードの番号を任意の数字に「読み替える」ことができる。

```typescript
// 番号変更可能な組み合わせ
const numberChangeRules: Record<Role, CardId[]> = {
  [Role.KIRA]: [0],     // デスノートのみ
  [Role.L]: [1],        // 逮捕のみ
  [Role.POLICE]: [2],   // 拳銃のみ
  [Role.WATARI]: [1],   // 逮捕のみ
  [Role.MISA]: [],      // なし
  [Role.MELLO]: [],     // なし
};
```

**重要な例外**: 拳銃カードで指名された場合、番号変更能力は使用不可。必ず本来の番号で最小カードを公開する。

### 5.3 初期手札配布ロジック

```typescript
function distributeInitialHands(playerCount: number, cardPool: Card[]): {
  hands: Card[][];
  deck: Card[];
} {
  // 1. デスノートと逮捕を1枚ずつ抜く
  const deathNote = cardPool.find(c => c.id === 0)!;
  const arrest = cardPool.find(c => c.id === 1)!;
  const remaining = cardPool.filter(c => c !== deathNote && c !== arrest);
  
  // 2. 残りから (人数×2 - 2) 枚をランダムに引く
  const drawCount = playerCount * 2 - 2;
  const shuffled = shuffle(remaining);
  const drawn = shuffled.slice(0, drawCount);
  const deck = shuffled.slice(drawCount);
  
  // 3. デスノート + 逮捕 + 引いたカード を混ぜて配布
  const toDistribute = shuffle([deathNote, arrest, ...drawn]);
  
  const hands: Card[][] = [];
  for (let i = 0; i < playerCount; i++) {
    hands.push([toDistribute[i * 2], toDistribute[i * 2 + 1]]);
  }
  
  return { hands, deck };
}
```

## 6. 勝利条件

### 6.1 キラ陣営の勝利条件（いずれか達成）

1. 裁きの時間にL全員を殺害
2. L陣営を残り1人まで減らす
3. すべての「逮捕」カードがゲームから除外される
4. メロがL全員を殺害した場合もキラ陣営の勝利

### 6.2 L陣営の勝利条件

- 捜査の時間に「逮捕」でキラを逮捕

### 6.3 メロの勝利条件

- 捜査の時間に「拳銃」でキラを殺害（単独勝利）

### 6.4 勝利判定ロジック

```typescript
function checkWinCondition(state: GameState): WinCondition | null {
  const alivePlayers = state.players.filter(p => p.isAlive);
  const aliveL = alivePlayers.filter(p => p.role === Role.L);
  const aliveLTeam = alivePlayers.filter(p => 
    [Role.L, Role.POLICE, Role.WATARI].includes(p.role)
  );
  
  // ゲーム内に残っている逮捕カードの数
  const remainingArrestCards = 
    state.deck.filter(c => c.id === 1).length +
    alivePlayers.flatMap(p => p.hand).filter(c => c.id === 1).length;
  
  // キラ陣営勝利条件
  if (aliveL.length === 0) {
    return WinCondition.KIRA_WINS;
  }
  if (aliveLTeam.length <= 1) {
    return WinCondition.KIRA_WINS;
  }
  if (remainingArrestCards === 0 && state.removedCards.some(c => c.id === 1)) {
    return WinCondition.KIRA_WINS;
  }
  
  // L陣営勝利・メロ勝利は即時判定（カード使用時に処理）
  
  return null;
}
```

## 7. 特殊処理ルール

### 7.1 山札が尽きた場合

捨て札をシャッフルして新しい山札にする。除外された逮捕カードは戻らない。

### 7.2 プレイヤー脱落時の処理

- 脱落者の手札は山札の一番下に戻す
- 正体カードは公開される
- 以降は観戦モード（神視点）に移行

### 7.3 ミサがキラを知るタイミング

ミサはゲーム開始時にはキラが誰か分からない。以下のタイミングでのみキラを視認できる：
- 裁きの時間
- 死神カード発動時

```typescript
function filterGameStateForPlayer(
  fullState: FullGameState, 
  playerId: string,
  isSpectator: boolean
): FilteredGameState {
  // 観戦者は全情報を見れる
  if (isSpectator) {
    return fullState;
  }
  
  return {
    ...fullState,
    players: fullState.players.map(p => ({
      id: p.id,
      name: p.name,
      isAlive: p.isAlive,
      handCount: p.hand.length,
      // 自分の情報のみ詳細
      ...(p.id === playerId ? {
        role: p.role,
        hand: p.hand,
      } : {
        role: p.isAlive ? null : p.role,
      }),
    })),
    // キラ・ミサチャットは該当者と観戦者のみ
    kiraMisaChat: (
      isKiraOrMisa(playerId, fullState) 
        ? fullState.kiraMisaChat 
        : null
    ),
  };
}
```

## 付録B: エッジケース一覧

| ケース | 処理 |
|--------|------|
| 手札が使用も捨札も不可 | 追加ドロー→最初のカードを山札に戻す |
| 山札が尽きる | 捨て札をシャッフルして山札に |
| キラがデスノート未所持で裁きの時間 | スキップ（誰も死なない） |
| キラが自分をデスノートに書く | 許可（偽名で生存可能） |
| 偽名使用済みで再度名前を書かれる | 死亡 |
| アリバイ使用済みで再度逮捕される | 逮捕成功、L陣営勝利 |
| メロがキラ以外を殺害 | メロ正体公開、対象脱落、ゲーム続行 |
| メロがL全員を殺害 | キラ陣営勝利 |
| 全逮捕カードが除外 | キラ陣営勝利 |
| ホストが切断 | 再接続待ち |
| プレイヤーが切断 | 再接続待ち |
| 復帰不可能な場合 | 強制終了ボタンで終了 |
